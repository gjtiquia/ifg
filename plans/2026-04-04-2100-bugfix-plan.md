## Root Cause Analysis

### Issue #2: Terminal Death Bug

**Symptoms:**
- On Enter, command not typed to terminal
- Cursor invisible
- Terminal unresponsive/dead

**Likely cause:**
The command is being printed to stdout while still in alternate screen buffer (or raw mode). The alternate screen buffer exit happens AFTER printing the command, so the command gets printed to a buffer that's immediately discarded.

**Current flow (buggy):**
1. User presses Enter
2. `return state.GetSelectedCommand()` from input loop
3. `main.go:78`: `fmt.Println(selectedCommand)"` - printed to alternate buffer
4. `defer term.Restore()` - exits alternate buffer (command lost!)
5. Also: cursor hidden and not shown

**Correct flow:**
1. User presses Enter
2. Get selected command
3. Exit alternate screen buffer FIRST
4. Show cursor FIRST
5. THEN print command to stdout

### Issue #3: Screen Clearing

**Current approach:**
- `ClearScreen()` uses `\x1b[2J` (clear entire screen)
- Preserves previous terminal content
- Works, but doesn't match fzf behavior

**FZF approach:**
- Uses alternate screen buffer (ESC [ ?1049h)
- Entire session happens in separate buffer
- On exit, original terminal content restored
- More polished UX

**Implementation:**
1. Enter alternate screen: `\x1b[?1049h`
2. Clear screen: `\x1b[2J`
3. Render UI in alternate buffer
4. Exit alternate screen: `\x1b[?1049l`
5. Print command AFTER exiting alternate buffer

### Fix #1: Terminal Restoration Order (Critical - Issue #2)

**File:** `internal/ui/tui.go`

**Changes:**
1. Add `EnterAlternateScreen()` function
2. Add `ExitAlternateScreen()` function
3. Remove `ClearScreen()` calls from render loop
4. Only clear ONCE when entering alternate screen

**Implementation:**
```go
func EnterAlternateScreen() {
    fmt.Print("\x1b[?1049h")  // Enter alternate screen
    fmt.Print("\x1b[2J")      // Clear it
    fmt.Print("\x1b[H")       // Move cursor to top-left
    HideCursor()
}

func ExitAlternateScreen() {
    ShowCursor()
    fmt.Print("\x1b[?1049l")  // Exit alternate screen
}
```

**File:** `main.go`

**Changes:**
1. Call `EnterAlternateScreen()` BEFORE input loop
2. Call `ExitAlternateScreen()` BEFORE printing command
3. Ensure proper defer order
4. Remove `HideCursor()/ShowCursor()` from defer (handled in Enter/Exit)

**Implementation:**
```go
// In main():
ui.EnterAlternateScreen()
defer ui.ExitAlternateScreen()

selectedCommand := runInputLoop(state, term)

ui.ExitAlternateScreen()  // Exit FIRST

if selectedCommand != "" {
    fmt.Println(selectedCommand)
    os.Exit(0)
}
```

### Fix #2: Drawing Format (Issue #1)

**File:** `internal/ui/tui.go`

**Changes:**
1. Add header rendering
2. Change prompt from "search:" to "type to search:"
3. Add separator "---" line
4. Render full entry blocks (title lines + description lines + command)
5. Apply '>' prefix to ALL lines of selected entry

**Implementation:**
```go
func Render(state *State) {
    // Clear alternate screen
    fmt.Print("\x1b[2J\x1b[H")

    // Header
    fmt.Println("\x1b[1mifg - [i] [f]or[g]ot that cmd again\x1b[0m")
    fmt.Println()

    // Search prompt
    prompt := "type to search: "
    if state.Mode == ModeNormal {
        prompt = "normal mode: "
    }
    fmt.Printf("%s%s\n", prompt, state.SearchBuf)
    fmt.Println()

    // Separator
    fmt.Println("---")
    fmt.Println()

    // Entries
    visibleHeight := state.TerminalHeight - 8  // Reserve space for header, prompt, etc
    for i := 0; i < visibleHeight && i+state.ScrollOffset < len(state.Filtered); i++ {
        entryIdx := i + state.ScrollOffset
        entry := state.Filtered[entryIdx]

        isSelected := entryIdx == state.SelectedIdx
        prefix := "  "
        if isSelected {
            prefix = "> "
            fmt.Print("\x1b[1m")  // Bold
        }

        // Title lines (comment lines starting with #)
        if entry.Title != "" {
            fmt.Printf("%s# %s\n", prefix, entry.Title)
        }

        // Description lines
        for _, desc := range entry.Description {
            fmt.Printf("%s# %s\n", prefix, desc)
        }

        // Command
        fmt.Printf("%s%s\n", prefix, entry.Command)

        if isSelected {
            fmt.Print("\x1b[0m")  // Reset
        }

        fmt.Println()  // Empty line between entries
    }

    // No results message
    if len(state.Filtered) == 0 {
        fmt.Println("\x1b[90mNo results found\x1b[0m")
    }
}
```

**Important note:** Title and Description are stored WITHOUT `#` in the config parser. When rendering, we ADD the `#` prefix. Command is rendered as-is (no `#`).
